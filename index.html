🔥 ตั้งค่า Firebase

เพื่อให้ข้อมูลซิงค์ระหว่างทุกเครื่อง ต้องใส่ข้อมูล Firebase ของคุณ

วิธีสร้าง Firebase (ฟรี):
1. เข้า console.firebase.google.com
2. กด Add project → ตั้งชื่อ → Create
3. กด Realtime Database → Create database → Start in test mode
4. คัดลอก URL เช่น https://xxx-default-rtdb.firebaseio.com
